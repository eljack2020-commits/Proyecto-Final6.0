Rutas recomendadas

Construimos caminos de aprendizaje para que avances paso a paso, desde los conceptos más básicos hasta habilidades avanzadas. Elige una ruta según tu interés y comienza a aprender a tu propio ritmo.

  • Full Stack: Aprende a desarrollar aplicaciones completas utilizando JavaScript tanto en el frontend como en el backend.
    • Fundamentos de HTML, CSS y JavaScript.
    • Uso de bases de datos con MongoDB y MySQL.
  • Frontend: Aprende a construir interfaces web interactivas y visualmente atractivas.
    • Fundamentos de HTML5, CSS3 y JavaScript.
    • Frameworks y librerías como React.js y Vue.js.
  • Backend: Aprende a construir la lógica del servidor y a trabajar con bases de datos y autenticación.
    • Desarrollo con Node.js y Express.
    • Conexión a bases de datos (SQL y NoSQL) como MongoDB, PostgreSQL y MySQL.
